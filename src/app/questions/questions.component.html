Questions

Imports

When we try to build Angular app we need several modules which are the part of different libraries, so we use import keyword to include those module in our file.

Declarations

When we start building angular application we divide the whole application in Components and we make different Components to work with them separately. Declarations array tells that which Component belongs to which module.

Providers

Providers are to defines services of different modules which need to work with dependency of that module.

Component

Component is a small part of a whole application like navbar, gallery etc. It a combination of HTML and a typescript class that provide metadata that decides how the component should processed.

Directives

Diectives are used to modify DOM, example like:- ngFor, ngIf Etc.

Models

Models is the MVC based application structure based term which is used to modeling the data used in the view.

Modules

Modules is a term that contains similar Components, directives, and services to make an application. An angular application at-least have one module to work with.

Services

Services are made for logic behind the view
